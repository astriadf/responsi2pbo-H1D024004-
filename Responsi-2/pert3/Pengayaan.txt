Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:
1. extends berfungsi untuk mewarisi semua atribut dan method dari class induk (parent class) ke class anak (child class), contohnya class Manajer extends Karyawan -> Manajer mewarisi nama dan gajiPokok dari Karyawan.

2. Overloading itu membuat method dengan nama sama tapi parameter berbeda, contohnya constructor BarangMusik dengan 2, 3, atau 4 parameter.
   Sedangkan overriding itu membuat method dengan nama dan parameter sama persis, tapi isi methodnya diganti di subclass, contohnya method tampilInfo() di Karyawan ditulis ulang di Manajer agar bisa menampilkan tunjangan.

3. super dipakai untuk memanggil constructor dari parent class. Contohnya super(nama, gajiPokok) di constructor Manajer memanggil constructor Karyawan.

4. Jika variabel di Karyawan diubah jadi private maka subclass (Manajer) tidak bisa langsung akses nama atau gajiPokok. Solusinya yaitu gunakan getter/setter (getNama(), getgajiPokok()) atau ubah access modifier jadi protected.

5. Kita perlu menggunakan inheritance supaya tidak menulis ulang atribut/method yang sma berkali-kali. Contoh: nama dan gajiPokok cukup ditulis sekali di Karyawan, lalu dipakai lagi di Manajer.
